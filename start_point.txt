## Project idea

You will build a **private email receiving system** for:

`anything@belfellah.tech`

Example:

`instagram@belfellah.tech`
`test123@belfellah.tech`
`support@belfellah.tech`

You do **not** create real mailboxes one by one.
You use **catch-all email**.

## Simple architecture

```text
Website sends verification email
        ↓
anything@belfellah.tech
        ↓
DNS MX record
        ↓
Postfix mail server
        ↓
Your Node/Django script
        ↓
Database
        ↓
React inbox panel
```

## What you need to learn first

### 1. DNS for email

Learn these:

```text
A record
MX record
TXT record
```

For receiving only, the most important is:

```text
MX record = tells the internet where your email server is
```

Example:

```text
belfellah.tech MX mail.belfellah.tech
mail.belfellah.tech A your_server_ip
```

---

### 2. Postfix

Postfix is the mail server.

It will receive emails for:

```text
@belfellah.tech
```

You configure it as **catch-all**, so every address works.

---

### 3. Pipe email to your app

Instead of saving email inside normal mailbox, Postfix sends the email to your script.

Example:

```text
Postfix receives email
→ sends raw email to script
→ script parses email
→ saves to database
```

Your script can be:

```text
Node.js
Python
Django
```

---

### 4. Database tables

Start with 2 tables.

**email_addresses**

```text
id
email
created_at
active
label
```

Example:

```text
instagram@belfellah.tech
```

**messages**

```text
id
to_email
from_email
subject
body
raw_email
received_at
```

---

### 5. API logic

Create email:

```http
POST /api/emails
```

Body:

```json
{
  "name": "instagram"
}
```

Result:

```json
{
  "email": "instagram@belfellah.tech"
}
```

Then inbox:

```http
GET /api/emails/instagram/messages
```

---

## Best first version

Do not start with sending emails.
Do only this:

```text
Receive email → save to database → show in dashboard
```

That is enough for verification codes.

## Tools I recommend

Use:

```text
Postfix = receive emails
Node.js or Django = API
PostgreSQL = database
React = dashboard
Docker = development
VPS = real public use
```

Your first goal:

**Make one email arrive to `test@belfellah.tech` and save it in your database.**
